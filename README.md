# Autism Support Application

A Flask-based web application designed to provide support and resources for autism awareness and community engagement.

## 📋 Overview

This is a simple yet effective web application built with Flask that serves as a platform for autism-related information and resources. The application includes a clean interface with template-based rendering for easy customization and extensibility.

## 🚀 Features

- Lightweight Flask web server
- Template-based HTML rendering
- Static file support for CSS, images, and JavaScript
- Easy-to-extend architecture
